Test Admin

The main navigation (left panel) includes:

  1. Tests
  2. Users

Overview of each option on panel​

S.ILeft panel optionsWhat it is / What you do there
1TestsCreate, manage, publish tests and view reports/results
2UsersAdd and manage test admin and candidate users

Overall workflow​

Step 1 — Tests Creation​

Primary Details​

  1. Go to Tests option on the left panel
  2. Click on Create Manually option
  3. Enter Test Name and Description details (for example — Feedback assessment for district officers)
  1. Select the tag types you want for the assessment — the drop down option will have the tags created by the system admin in your organisation (For example — Difficulty level, focus area, audience, assessment type etc)
  2. Under tags based on the tag types you selected drop down options will be available for you to choose (For example — If you have selected difficulty level as the tag type — the options added by your system admin such as hard, medium and easy will be shown)

Select Questions (Auto / Manual)​

Auto selection of questions

  1. You have the option to choose the no of questions to randomly pull from each of the tags that you selected in the previous step — Easy in difficulty level, Pedagogy in focus area

Manual selection of questions

  1. You have the option to select questions manually from the question bank which will show all the questions that have been created for each tag type
  2. You can choose the required questions and add it to the test.

Test Configuration​

Test Schedule

  1. In this box, you basically choose the start time and end time for the test to be active.

Candidate Experience

  1. For pre test guidelines, you can add brief instructions on how to take the test (internet, time limit, do's & don'ts).
  2. For the test completion message, you can add a short note confirming submission and what happens next (results/next steps).
  3. You can set the language in which candidates will view the assessment portal.
  4. With the candidate information form, we can select the form that should be shown to candidates before they start the test to collect the required candidate details. These forms can be created and managed using form management from side-bar.
  5. Under the Certificate dropdown, you can select the certificate template created by the System Admin. You will only be able to choose from the available certificate templates and will not be able to edit the certificate design or content.

Test Rules

  1. You can choose to shuffle questions for each candidate to reduce copying.
  2. You can choose to show marks during the test for real-time visibility.
  3. You can choose to display results immediately after the test is completed.

Step 2 — Publish Test​

Once you click the Save option, the test will be saved under the Tests page. You can view it anytime by clicking on the Tests option on the left panel.

From there, you can:

  • Download the QR code to share with candidates
  • View reports and results once candidates start attempting the test
  • Edit the test details and configuration if needed
