# Sashakt Platform Walkthrough

This guide provides a walkthrough of the Sashakt platform for different administrative roles. Use the sections below to navigate directly to the role-specific documentation and understand the platform's features and functionalities.

## Available Roles

- System Admin
- State Admin
- Test Admin

---

# System Admin

The System Admin has access to all platform features and can manage assessments, users, organization settings, and reports.

## Main Navigation Menu

The main navigation panel is available on the left side of the platform.

| Sr. No. | Navigation Menu | What You Can Do |
|----------|----------------|-----------------|
| 1 | **Tests** | Create, manage, publish tests, and view reports and results. |
| 2 | **Test Templates** | Create reusable test structures based on tags or question sets. |
| 3 | **Question Bank** | Add, edit, and manage questions manually or through CSV uploads. |
| 4 | **Tag Management** | Create and organize tags used to categorize questions and tests. |
| 5 | **Forms** | Create and manage forms to collect candidate details before starting a test. |
| 6 | **Certificates** | Create and manage certificate templates for candidates who complete tests. |
| 7 | **Users** | Add and manage users with different roles such as State Admins and Test Admins. |
| 8 | **My Organisation** | Configure organization settings including organization name, logo, test rules, and platform labels. |
